# Commandes de déploiement

## 1. Déploiement Firebase Hosting (Frontend)
# Option A: Script npm (recommandé)
npm run deploy:hosting

# Option B: Commandes manuelles
cd frontend
npm run build
cd ..
firebase deploy --only hosting
